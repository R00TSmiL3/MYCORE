## 🎯 Getting Started

### Prerequisites
- Python 3.8+
- Modern web browser
- Git

### Installation

1. Clone repository:
```bash
git clone https://github.com/R00TSmiL3/MYCORE.git
cd MYCORE
```

2. Setup (jika ada backend Python):
```bash
pip install -r requirements.txt
```
